# Portafolio Personal - Matías Leonardo Retamal Barrera

Este es el repositorio del portafolio personal de Matías Leonardo Retamal Barrera, Desarrollador de Software y Especialista en Soporte TI.

El sitio web está diseñado con un estilo minimalista de "Galería de Arte" para destacar los proyectos y habilidades profesionales.

## Tecnologías Utilizadas

*   **HTML5**: Estructura semántica.
*   **CSS3**: Diseño moderno con Flexbox y CSS Grid. Sin frameworks pesados.
*   **JavaScript (ES6+)**: Navegación "sticky" y desplazamiento suave.

## Estructura del Proyecto

*   `index.html`: Página principal.
*   `styles.css`: Hoja de estilos.
*   `script.js`: Lógica de interacción.

## Cómo ver el proyecto

Simplemente clona este repositorio y abre el archivo `index.html` en tu navegador web.
